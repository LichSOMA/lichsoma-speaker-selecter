# LichSOMA's Speaker Selecter

> **이 모듈은 생성형 AI의 도움으로 제작되었습니다.**

## 소개

Foundry VTT의 채팅 시스템을 확장하는 모듈입니다. 토큰 선택과 독립적인 스피커 선택, 채팅 커스터마이징, 나레이터 모드 등 다양한 기능을 제공합니다.

## 주요 기능

### 🎭 스피커 선택
- 토큰 선택과 독립적으로 말하는 캐릭터를 선택할 수 있습니다
- 채팅 입력 창 상단의 드롭다운 메뉴에서 스피커 선택
- OOC(Out of Character) 모드 지원

### ⚙️ 스피커 설정
- 채팅 입력 창의 톱니바퀴 버튼을 클릭하여 사용 가능한 액터 관리
- 드래그 앤 드롭으로 액터를 추가/제거
- 이름 또는 태그로 검색 가능

### 😊 감정 시스템
- 각 액터별로 감정 표정을 설정하고 관리
- 채팅 시 원하는 감정을 선택하여 포트레잇 변경
- 감정 관리 버튼을 통해 추가/편집

### 🎨 채팅 커스터마이징
- **포트레잇**: 크기 조절 및 표시 옵션
- **폰트**: 헤더와 메시지의 폰트, 크기, 굵기 설정
- **한자 전용 폰트**: 한자에만 적용되는 별도 폰트 설정
- **유저 색상**: 발신자 이름에 유저별 색상 적용
- **채팅 머지**: 연속된 같은 스피커의 메시지에서 헤더 숨김
- **Out of Character**: 채팅 메시지를 '@'로 시작할 경우 무조건 OOC 처리
- **마크다운 문법**: 채팅 메시지에서 마크다운 스타일 포맷팅 사용 가능
  - `~텍스트~`: ~~취소선~~
  - `*텍스트*`: *이탤릭*
  - `**텍스트**`: **볼드**
  - `***텍스트***`: ***볼드+이탤릭***
  - `[[텍스트|루비]]`: 루비 문자

### 📖 나레이터 모드 (GM 전용)
- 채팅 입력 창의 나레이터 버튼 클릭으로 활성화
- 타이핑 효과와 함께 메시지 표시
- 타이핑 속도, 사운드, 스타일 커스터마이징 가능
- 나레이터 채팅 카드 옵션으로 중앙 정렬 및 이탤릭 효과 적용
- 'desc '로 시작할 경우 나레이터 바를 활성화 하지 않아도 나레이터 채팅 카드로 처리

### 💾 채팅 로그 내보내기
- 채팅 로그를 HTML 파일로 저장
- 이미지를 경로 링크 또는 Base64로 저장 선택 가능
- 커스텀 CSS 적용으로 스타일 변경 가능
- 주사위 롤 상세 정보 표시 옵션

### 📚 채팅 로그 아카이브
- 과거 채팅 로그 조회 및 검색
- 폴더 구조로 정리된 로그 탐색
- 로그 내 키워드 검색 기능

### ⌨️ 타이핑 표시
- 다른 사용자가 채팅을 입력 중일 때 실시간 표시

## 설치 방법

1. Foundry VTT의 모듈 관리에서 "Install Module" 클릭
2. Manifest URL 입력 또는 검색을 통해 설치
3. 게임 월드에서 모듈 활성화

## 설정

게임 설정(Game Settings) > 모듈 설정(Module Settings)에서 다양한 옵션을 조정할 수 있습니다.

### 주요 설정 항목
- 포트레잇 표시 및 크기
- 채팅 폰트 및 스타일
- 나레이터 모드 설정
- 채팅 로그 내보내기 경로
- 채팅 머지 활성화

## 사용 팁

- **스피커 빠른 변경**: 스피커 드롭다운에서 자주 사용하는 캐릭터를 선택하세요
- **감정 표현**: 채팅 전 감정을 선택하여 캐릭터의 상태를 표현하세요
- **나레이터 연출**: 중요한 장면 연출 시 나레이터 모드를 활용하세요
- **로그 보관**: 세션 종료 후 채팅 로그를 HTML로 저장하여 보관하세요
